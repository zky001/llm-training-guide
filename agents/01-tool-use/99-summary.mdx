# A1 · 小结与自测

## 一图回顾

```mermaid
flowchart LR
    S["📋 工具清单<br/>（说明书入上下文）"] --> D["🤔 模型决策<br/>（用不用？用哪个？）"]
    D -->|"需要工具"| J["📝 生成调用 JSON<br/>（这也是生成文本）"]
    J --> E["⚙️ 外部程序执行"]
    E --> R["👀 结果拼回上下文"]
    R --> D
    D -->|"不需要工具"| A["✅ 直接回答"]
```

一句话收束：**模型没有手——「调用工具」是生成一段结构化文本，执行权在外部程序**；而读懂说明书、递对纸条、判断该不该伸手，这整套本事是预训练、SFT 和 RL 一层层喂出来的。
