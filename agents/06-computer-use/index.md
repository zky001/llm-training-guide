# A6 · 智能体上电脑

前五章里，智能体用的工具都是别人写好、封装成 API 递过来的（A1）。可现实里**大部分软件根本没有 API**：老旧的政务系统、桌面软件、别人家的网页。要让智能体真正干活，得让它坐到电脑前，用人类的方式操作电脑——敲终端、改代码、点鼠标。

这一章分两段。先看**代码智能体**——所有智能体里最成熟、最先商用的一类，2024 年以来最出圈的产品几乎都在写代码；再看更难、还在爬坡的**计算机使用**。你会发现，两者成熟度的天壤之别，可以用一个词解释：能不能自动判对错。
