# 第 0 章 · 小结与自测

恭喜你走完预备知识！四节内容其实讲的是同一个故事的四个侧面：

| 小节 | 核心直觉 | 在大模型中的角色 |
| --- | --- | --- |
| [0.1 什么是机器学习](./01-what-is-ml.mdx) | 模型 = 带参数的函数；学习 = 自动调参数让偏差变小 | 定义了「训练」这件事本身 |
| [0.2 向量与矩阵](./02-vectors-matrices.mdx) | 万物先变成向量；矩阵是空间的变形器 | 模型的输入输出与每一层的运算形态 |
| [0.3 概率](./03-probability.mdx) | 生成 = 掷一枚由上文决定的特制骰子；估准概率需要海量数据 | 模型输出的数学形态与数据饥渴的根源 |
| [0.4 导数与梯度](./04-gradient.mdx) | 梯度指路，学习率定步长，一步步走向损失的谷底 | 训练的发动机：更新参数的具体算法 |

串成一句话：**大模型是一个把向量层层变形的巨型函数，它输出「下一个词」的概率分布，训练就是用梯度下降调整矩阵里的几千亿个参数，让它对海量文本的预测偏差越来越小。**
