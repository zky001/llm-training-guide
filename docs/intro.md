# 导读：如何使用本教程

欢迎！这是一份**从零开始、讲透大模型是怎么训练出来的**中文交互式教程。

读完全书，你将能回答这些问题：

- ChatGPT / Claude / Qwen 这类大模型，本质上在做一件什么事？
- 「训练一个大模型」具体是在算什么？为什么要几千张 GPU、烧几亿人民币？
- 预训练、SFT、RLHF……这些黑话分别指生产流水线的哪一段？
- 为什么大模型会一本正经地胡说八道（幻觉）？为什么每次回答不一样？

## 🎚️ 双轨阅读：一本书，两种读法

本站每一节内容都分成两层，这是全站最重要的功能：

- **直觉层（默认显示）**：用类比、动画和交互实验讲清楚「为什么」，只需要初中代数就能读懂。
- **深入层（折叠在「🔬 深入一层」块里）**：严格定义、数学推导、代码实现，面向学过微积分和线性代数的读者。

两种打开方式：

1. **逐个点开**：遇到感兴趣的「🔬 深入一层」就点开看。
2. **一键切换**：点击页面**右下角的悬浮按钮**，在「🧠 直觉模式」和「🔬 深入模式」之间切换，深入模式会自动展开全站所有推导。

:::tip 给不同读者的建议
- 🎒 **高中生 / 零基础**：保持直觉模式，按章节顺序读，把每个 🎮 实验都玩一遍——实验比文字更重要。
- 🎓 **大学生**：切到深入模式，推导自己先推一遍再看答案，章末自测全做。
- 💼 **工程师**：可以按需跳章，交互实验也很适合拿去给别人做演示。
:::

## 🎮 交互实验

全书有大量可以动手玩的实验组件：拖动小球体验梯度下降、亲手训练一个 BPE 分词器、掷「语言骰子」看模型如何生成文字……

**强烈建议把每个实验都玩到「玩坏」为止**——比如故意把学习率调到最大看训练爆炸。亲手把它玩坏一次，胜过读十遍文字。
