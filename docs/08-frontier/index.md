# 第 8 章 · 前沿与全景

前七章讲完了「怎么炼」：数据怎么洗、损失怎么降、GPU 怎么切、偏好怎么对齐、推理怎么提速。最后一章换个问题——**往哪去？**

我们挑了四个正在重塑大模型的方向，每个都能用前面学过的零件看懂：

1. **MoE**——参数堆到万亿，账单却只按百亿付（8.1）；
2. **长上下文**——从半篇论文到一次读几本书（8.2）；
3. **多模态**——图像、语音也切成 token（8.3）；
4. **推理模型**——用强化学习教模型「多想一会儿」（8.4）。

最后的 8.5 是全书大结局：一张可下载的全景大图，把九章内容收进一页纸。

:::
