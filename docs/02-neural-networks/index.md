# 第 2 章 · 神经网络与训练三件套

第 1 章说，语言模型是一个「上文 → 下一个词概率」的函数。但这个函数**内部**长什么样？「错」怎么量化成一个数？几千亿个参数又是怎么自动调的？

这一章把黑箱拆开。你会看到，训练任何神经网络（包括 GPT）都只需要三件套：

1. **模型**——一个由神经元拼成的、足够能弯的函数（2.1、2.2）；
2. **损失**——给「错得多离谱」定价的评分标准（2.3）；
3. **引擎**——梯度下降负责调参数，反向传播负责算梯度（2.4、2.5）。

最后在 2.6 节，三件套合体：你将在浏览器里亲手训练一个真实的神经网络。

| [2.2 ReLU 积木](./02-layers.mdx) | 折角够多，能拼出任何曲线（万能逼近） | 🎮 ReLU 积木 |
