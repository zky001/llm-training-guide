# 第 2 章 · 小结与自测

## 全章鸟瞰

这一章拆开了「训练」这个黑箱。任何神经网络——从本章 25 个参数的玩具到几千亿参数的 GPT——都在跑同一个循环：

```mermaid
flowchart LR
    A["数据<br/>（抽一小批）"] --> B["前向<br/>算出预测"]
    B --> C["损失<br/>给偏差打分"]
    C --> D["反向传播<br/>算出全部梯度"]
    D --> E["更新参数<br/>θ ← θ − η × 梯度"]
    E --> A
```

三件套各司其职：**模型**提供一个足够能弯的函数（神经元的折角拼出万能逼近）；**损失**把「错得多离谱」压成一个可比较的数字；**引擎**里梯度下降决定往哪走、反向传播负责把几千亿个「往哪走」一次算齐。循环亿万次，损失曲线缓缓落地，模型就「学会」了。

## 要点回顾

| 小节 | 两行要点 |
| --- | --- |
| [2.1 一个神经元](./01-neuron.mdx) | 神经元 = 加权求和 + 偏置 + 激活函数。没有激活函数，叠一万层仍是直线；ReLU 给直线一个折角。 |
| [2.2 ReLU 积木](./02-layers.mdx) | 一个神经元一个折角，折角够多能贴合任何连续曲线（万能逼近）。容量不足 = 欠拟合的根源。 |
| [2.3 损失函数](./03-loss.mdx) | 回归用 MSE，分类和语言模型用交叉熵：给正确答案的概率越小，罚分越重。损失定义了「学什么」。 |
| [2.4 梯度下降进阶](./04-gradient-descent.mdx) | 沟壑地形逼出之字形；每步抽一小批数据 = SGD，噪声常是帮手；动量与 AdamW 驯服沟壑。 |
| [2.5 反向传播](./05-backprop.mdx) | 口诀「下游梯度 = 上游梯度 × 本地导数」沿计算图反向接力，一趟拿到全部参数的梯度。 |
| [2.6 训练实验室](./06-training-lab.mdx) | 四步循环点火实测：心电图前陡后缓为正常；容量太小欠拟合，训过头过拟合，靠验证集把关。 |

一句话浓缩全章：**训练 = 用反向传播算出的梯度，沿损失面反复下山，直到模型这个「折角函数」贴住数据的规律。**
