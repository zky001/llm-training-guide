# 第 3 章 · Transformer 架构

第 2 章造出的神经网络已经是个「万能函数」，但它有个致命短板：**孤立地处理每个输入**，既看不懂词的先后顺序，也不会让「它」回头去找「小猫」。语言恰恰活在上下文里。

2017 年，Transformer 用一个漂亮的机制解决了这件事——**注意力**：让每个词自己决定该看谁、看多少。从 GPT 到 Claude 到 Qwen，2025 年所有主流大模型用的都是这副骨架。这一章我们把它拆开看个明白：
