# 第 7 章 · 评测：怎么知道模型好不好

训练的时候，我们盯着损失曲线（[2.3 节](../02-neural-networks/03-loss.mdx)）——但用户不关心损失。用户关心的是：**这个模型好不好用？比那个强吗？**

这个听起来天经地义的问题，实际上是大模型领域最泥泞的问题之一：

- **考题会泄露**——评测题混进训练集，模型是「背过原题」而不是「会做题」；
- **裁判有偏心**——不管是人还是模型当评委，都偏爱更长、更自信、更漂亮的回答；
- **榜单能刷分**——针对性优化几个热门基准，排名可以很好看，实际体验未必。

所以这一章的目标不是给你一张「最强模型排行榜」，而是给你一副**读榜单的怀疑眼镜**：知道每个分数是怎么来的、能信几分、坑在哪里。
