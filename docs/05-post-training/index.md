# 第 5 章 · 后训练与对齐：从「会说话」到「说人话」

第 4 章烧掉几个亿炼出来的<Term id="base-model">基座模型</Term>，是一位「最强接话茬选手」：它读过整个互联网，语法完美、知识渊博。但你问它「中国的首都是哪里？」，它很可能给你续写出十个类似的问题——因为在互联网上，一个问题后面最常跟着的往往是**更多的问题**，而不是答案。

它不是不知道，它只是在做预训练教它的唯一一件事：**把最像的文本接下去**。

后训练（post-training）的使命，就是把这位选手调教成真正的助手——听得懂指令、答得上问题、拒得掉危险请求。这一章讲完这条流水线的每一环：
