# 延伸资料清单

按主题组织。难度标记：🟢 零基础可看 · 🟡 需要本书前几章基础 · 🔴 面向进阶读者。

## 可视化与视频

- 🟢 [3Blue1Brown 神经网络系列](https://space.bilibili.com/88461692)（B 站官方号）——梯度下降、反向传播、Transformer 的顶级动画讲解。
- 🟢 [LLM Visualization (bbycroft.net)](https://bbycroft.net/llm)——3D 漫游一个真实 GPT 的每一层计算。
- 🟡 [Jay Alammar: The Illustrated Transformer](https://jalammar.github.io/illustrated-transformer/)——被引用最多的 Transformer 图解。
- 🟡 [Karpathy: Deep Dive into LLMs like ChatGPT](https://www.youtube.com/watch?v=7xTGNNLPyMI)——3.5 小时讲完从预训练到 RLHF 的全流程。

## 系统课程

- 🟡 [动手学深度学习 d2l.ai](https://zh.d2l.ai/)——李沐团队的中文教材，代码齐全。
- 🟡 [Hugging Face NLP 课程](https://huggingface.co/learn/nlp-course/zh-CN/)（有中文版）。
- 🔴 [Stanford CS336: Language Modeling from Scratch](https://stanford-cs336.github.io/)——从零实现大模型训练全栈。
